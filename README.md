# Instructions for starting a project:

``` bash
    1.  git clone https://github.com/Roman-Gulamov/axiosform
```

``` bash
    2.  yarn install
```

``` bash
    3.  yarn build
```

``` bash
    4.  http://localhost/axiosform/build/
```
